# 安装与配置

> **本页职责**：把游戏跑起来、填好驱动 GM 的模型凭据。

## 怎么装

现在的玩法是自己起服务、用浏览器玩:

```bash
git clone <本项目地址>
cd dicelore
npm install
```

装完依赖后,开两个终端分别起后端和前端。

后端(会自动懒建存档,不需要提前初始化数据库):

```bash
PORT=8787 npm run dev -w @dicelore/backend
```

前端:

```bash
npm run dev -w @dicelore/frontend
```

前端起来后终端会打印一个本机地址,浏览器打开它就能看到首页。

## 填 API key + baseURL

GM 由一个大模型驱动。打开客户端的**配置**页(齿轮图标),在"模型连接"一节里填:

- **模型 / Agent 底座**:选哪个模型当 GM、走哪套运行时驱动它;
- **API key**:你的模型访问凭据——只存在你本机,不会被上传;
- **baseURL**:模型服务地址,留空用默认地址即可。

填好后客户端就会用它驱动 GM。想自检有没有填对,点旁边的"连接测试"——它能分辨出"没连上"和"key/权限不对"这两种失败。
