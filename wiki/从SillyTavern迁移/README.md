# 从 SillyTavern 迁移

> **本页职责**：给从 SillyTavern（万域界一类预制角色卡/世界书团本）转过来的人指路——按身份挑一篇看。

如果你之前在 SillyTavern 里玩过、或者做过角色卡/世界书，dicelore 这边有很多概念能对上号，但也有几处关键差异需要提前知道。本文件夹只讲"怎么把你熟悉的东西对应过来、怎么迁移"，不是架构层面的深度技术文档（那部分留给开发者看，见下）。

## 按身份挑一篇

| 你是 | 看这篇 | 讲什么 |
|------|--------|--------|
| 只是想玩，之前用过 SillyTavern 角色卡/世界书 | [玩家侧](玩家侧.md) | 你熟悉的东西在 dicelore 里叫什么、体验上会有什么不一样 |
| 做过 SillyTavern 角色卡/世界书，想把内容搬过来自己造团本 | [作者侧](作者侧.md) | 概念对照表 + 具体迁移步骤 |

## 面向开发者的深度对比

如果你想搞清楚两边架构设计上的差异（GM 决策链怎么落地、状态怎么更新、可借鉴/不可借鉴什么），那是研究性质的内容，不在本文件夹——见 [`docs/research/silly_tavern/`](../../docs/research/silly_tavern/)。
