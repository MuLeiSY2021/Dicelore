# 贡献者公约行为准则

## 我们的承诺

为了营造开放、友善的环境，我们作为贡献者和维护者承诺：无论年龄、体型、可见或不可见的残疾、族裔、性征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性取向如何，参与本项目和社区的每个人都不会受到骚扰。

我们承诺以有助于建立开放、热情、多元、包容和健康社区的方式行事和互动。

## 我们的准则

有助于为我们的社区营造积极环境的行为包括但不限于：

* 对他人展示同理心和善意
* 尊重不同的意见、观点和经验
* 给予并得体地接受建设性反馈
* 承担责任并向受我们错误影响的人道歉，并从经验中学习
* 不仅关注于我们个人，更关注于整个社区的最大利益

不当行为包括但不限于：

* 使用与性有关的言语或图像，以及任何形式的性关注或挑逗
* 嘲弄、侮辱或贬低性评论，以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可，发布他人的私人信息，例如物理地址或电子邮箱地址
* 在专业场合中可以被合理认定为不当的其他行为

## 执行责任

社区维护者有责任解释和落实我们所认可的行为准则，并会对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

社区维护者有权利和责任删除、编辑或拒绝与本行为准则不相符的评论、提交、代码、wiki 编辑、议题或其他贡献，并在适当时机解释采取措施的理由。

## 适用范围

本行为准则适用于本项目的所有社区场合，也适用于在公共场合代表本项目或其社区时的个人行为。代表本项目或社区的情形包括但不限于：使用项目官方电子邮箱、通过官方社交媒体账户发帖，或在线上或线下活动中担任指定代表。

## 执行

如遇辱骂、骚扰或其他不当行为，请通过 GitHub 私密渠道联系项目维护者 **[@MuLeiSY2021](https://github.com/MuLeiSY2021)** 举报（可经仓库 **Security → Report a vulnerability** 私密通道，或私信维护者）。所有投诉都将得到及时且公正的审查与处理。社区维护者有义务为事件举报者保密。
